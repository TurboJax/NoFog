# NoFog

Simply removes fog from the game.  

Supports 1.17 to 1.21.11.  
**Versions 1.17-1.19.4 say that they are for a newer version of minecraft.  They still work**  

### This is incompatible with Sodium!
Sodium doesn't work with resource packs that affect shaders.
Instead, I'd recommend installing [Reese's Sodium Options](https://modrinth.com/mod/reeses-sodium-options) and using that to disable fog.
